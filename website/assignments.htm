The assignments in this course are centered around the creation of a rendering engine that will generate realistic images.  The goal is to expose you to all the fundamental knowledge required to generate realistic 3D images.  Rendering is now a consumer hardware commodity.  While you may never write any other rendering code, the knowledge is fundamental in your use of other graphics packages (OpenGL, Direct3D, Java3d, etc), or rendering and animation systems (Alias/Wavefront, Softimage, 3D Studio Max, Maya, etc.).

Your focus should be on learning the concepts, discovering common patterns, and recognizing how to group and package the functional elements of rendering for maximum flexibility and extensibility.  I've created an example framework in Java that we will use for this class.

On a weekly basis, the assignments should take several (3 to 4) hours and should build upon the work of the previous week - don't get behind, it may be hard to catch up.  You may work in the language of your choice (C, C+, Java, Fortran) on the platform of your choice (Windows, Unix, Linux).  I will be posting solutions implemented in   Java.  The Java choice is because tools are freely available for all platforms, the component model is much easier to implement,  documentation is inline, multi-threading is easier to implement, and with a little care in programming there is negligible performance penalty as compared to C or C++.
